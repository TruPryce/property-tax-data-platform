# Airflow DAGs

This area contains thin Airflow 3.3 inbound adapters. DAGs schedule application use cases and exchange release IDs or object URIs; they do not implement county parsing, database SQL, or business rules.

The initial DAG implementation is tracked by task 5.2 in the [foundation change](../openspec/changes/bootstrap-six-county-appraisal-platform/tasks.md).
